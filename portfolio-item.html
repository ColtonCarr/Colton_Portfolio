Tic-Tac-Toe

A stepping stone

Building Tic-Tac-Toe was the entryway into learning Unity as well as C#. In this project, I began using Unity's UI to create buttons and grids that would change color as well as fill with the appropriate player's icon.

The code allows player 1 to select their preferred choice of icon. Then, after selecting the desired grid, player 2 would place theirs. The game would continue until either player would win or spaces would run out.
